📢 공지사항 (상단 티커)

여기 등록한 공지는 메인 화면 상단에 흐르는 띠(티커)로 계속 노출됩니다. 활성/비활성·삭제로 관리하세요.

사용자 추가

충전은 사용자 추가 후 사용자 목록의 ‘+ 충전’ 버튼으로 진행하세요.

사용자 목록

베타 기능

베타 기능을 켜고/끄고, 지정한 테스터 계정에게만 노출합니다. 관리자는 항상 사용 가능. 여러 베타를 동시에 운영할 수 있습니다.

보고서 생성 로그
