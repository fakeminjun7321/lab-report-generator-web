# 보고서 예시 파일 두는 곳

이 폴더에 생성한 보고서 예시 파일을 넣으면, 예시 페이지(`/examples.html`)의
"보고서 생성 예시" 카드가 **자동으로** 다운로드 버튼을 띄웁니다.
(페이지가 열릴 때 아래 파일명이 있는지 검사해서, 있으면 버튼으로 바뀝니다. 코드 수정 불필요.)

## 파일명 규칙 (이 이름 그대로 저장)

| 보고서 종류 | 파일명 |
| --- | --- |
| 화학 사전보고서 | `chem-pre.docx` / `chem-pre.hwpx` |
| 화학 결과보고서 | `chem-result.docx` / `chem-result.hwpx` |
| 물리 결과보고서 | `phys-result.docx` / `phys-result.hwpx` |

- `.docx`, `.hwpx` 둘 중 하나만 넣어도 되고, 둘 다 넣으면 버튼이 둘 다 나옵니다.
- 다른 이름으로 저장하면 자동 감지가 안 됩니다. 위 이름을 정확히 지켜 주세요.

## ⚠️ 개인정보 주의

이 페이지는 **누구나 볼 수 있는 공개 페이지**입니다.
예시 파일에는 **실명·학번 등 개인정보가 없어야** 합니다.

- 보고서를 만들 때 이름/학번 칸은 비우거나 `예시` 같은 가짜 값으로 채우세요.
- 이미 만든 보고서를 쓸 경우, 표지·머리말의 이름과 학번을 지운 뒤 저장해 올리세요.
