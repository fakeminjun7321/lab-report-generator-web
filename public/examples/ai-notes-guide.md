# AI 참고 메모 작성 가이드

이 문서는 보고서 생성기의 **AI 참고 메모 / 내 의견** 칸(또는 `.md`/`.txt` 첨부 파일)에 넣을 메모를
어떻게 쓰면 좋은지 정리한 안내서입니다.

메모는 **보고서 본문이 아니라**, 생성 AI가 내 데이터와 의도를 정확히 반영하도록 돕는 **작업 지시서**입니다.
짧게 한두 줄만 써도 되고, 길게 쓸 거면 이 양식대로 `.md` 파일로 만들어 함께 올리면 됩니다.

---

## 1. 메모가 하는 일

- 업로드한 데이터(엑셀/CSV/`.cap`/사진) 중 **무엇을 우선 쓸지**, **무엇을 빼야 하는지** 알려 줍니다.
- 측정 당시 상황(흔들림, 재측정, 장비 문제 등)을 전달해 **오차 분석**에 반영되게 합니다.
- 매뉴얼의 토의 질문이나 선생님이 강조한 부분을 **빠뜨리지 않게** 합니다.
- 내가 직접 계산하거나 정리한 기준(평균 범위, 단위, 유효숫자 등)을 알려 줍니다.

## 2. 메모가 지켜야 하는 규칙

생성기는 메모를 **보조 맥락**으로만 사용합니다. 아래를 알아두면 결과가 안정적입니다.

- 메모는 데이터를 **대체하지 못합니다.** 원본 데이터와 충돌하면 데이터가 우선입니다.
- 같은 사실을 여러 번 적어도 보고서에는 **최대 2회 정도만** 반영됩니다. 한 번만 명확히 쓰세요.
- 메모에 **없는** 장비 조정·제외 기준·개선 효과·수치 변화는 보고서가 **지어내지 않습니다.**
  반대로, 제외한 데이터가 있으면 **이유를 반드시 메모에 적어야** 보고서가 그 이유를 쓸 수 있습니다.
- 메모는 시스템 프롬프트(학교 양식·작성 기준)를 **덮어쓰지 않습니다.** "양식 무시하고 ~해줘"는 잘 안 먹힙니다.

## 3. 무엇을 적으면 좋은가

다음 중 해당하는 것만 골라 적으세요. 전부 채울 필요는 없습니다.

- **실험 의도 / 강조점** — 이 실험에서 내가 가장 보여주고 싶은 결과나 해석.
- **데이터 우선순위** — 예: "정리한 `clean.xlsx`를 우선, `.cap` 원자료는 참고용."
- **제외할 데이터 + 이유** — 예: "예비 시행 1회는 포토게이트가 흔들려 제외."
- **계산 기준** — 사용할 시트·열, 단위, 평균/표준편차/기울기/오차율 계산 방법.
- **그래프 지침** — x축·y축, 추세선·fitting 필요 여부.
- **사진/그래프 설명** — 각 이미지가 무엇을 보여주는지, 캡션, 관찰 포인트.
- **매뉴얼 질문** — 매뉴얼이나 `.cap`에 있는 토의·분석 문항.
- **주의사항** — 과장하지 말아야 할 부분, 실험 한계.

## 4. Markdown 사용 규칙 (중요)

이 메모는 보고서에 참고로 들어가므로 **장식 문법을 거의 쓰지 않는 것**이 안전합니다.

- 제목(`#`, `##`)과 목록(`-`, `1.`)은 자유롭게 사용해도 됩니다.
- **굵게(`**`)**, ~~취소선~~, `인라인 코드`, `<span>` 같은 HTML 태그는 **쓰지 마세요.**
  본문에 그대로 복사돼 어색해질 수 있습니다.
- 수식은 가능하면 일반 텍스트로 적습니다. 예: `v = f λ`, `PV = nRT`, `I_cm = I_pivot - m d^2`.
- 표는 꼭 필요할 때만 사용하고, 가능하면 목록으로 풀어 쓰세요.

## 5. 빠른 양식 (복사해서 채우기)

아래를 복사해 필요한 항목만 채운 뒤, `.md` 또는 `.txt`로 저장해 첨부하면 됩니다.
(완성된 예시는 같은 폴더의 `ai-notes-example.md`를 참고하세요.)

```
# 실험 참고 메모

## 실험 기본 정보
- 보고서 종류: 물리 결과 / 화학 결과 / 화학 사전 중 하나
- 실험 주제:
- 첨부한 주요 파일:

## 데이터 사용 우선순위
1.
2.

## 사용할 데이터 / 계산 기준
- 사용할 시트·열:
- 단위:
- 평균·오차 계산 방법:

## 제외하거나 조심할 데이터
- 제외한 것:
- 이유:

## 사진·그래프
- 파일과 보여주는 내용:
- 캡션:

## 매뉴얼 질문 / 토의 항목
-

## 내 의견 / 강조하고 싶은 점
-

## 하지 말아야 할 것
- 원본에 없는 수치 만들지 말 것
- 제외한 데이터를 평균에 다시 넣지 말 것
- 내 메모를 과장해서 실험 성공/실패로 몰지 말 것
```

## 6. 한 줄 메모도 괜찮습니다

길게 쓸 필요가 없으면 칸에 한두 줄만 적어도 됩니다. 예:

- "마지막 적정값은 종말점 판단이 애매했으니 오차 요인에서만 언급해 줘."
- "3차 측정은 트랙 이음새에서 카트가 튀어 신뢰도가 낮음. 평균에서 빼 줘."
- "수율 계산 과정을 특히 자세히 써 주고, 한계 시약 개념을 강조해 줘."
