네이버 블로그에서 사용 팁과 새 기능 소식을 확인하세요.

블로그 보기

0. 시작하기

  • 로그인: 발급받은 이름·비밀번호로 로그인합니다. 만 14세 이상, 약관 동의가 필요합니다.
  • 크레딧: 보고서 종류별(사전/결과)로 크레딧이 차감됩니다. 화면의 예상 비용은 추정치이며, 실패한 생성은 원칙적으로 차감하지 않습니다. 크레딧은 현재 운영자가 수동 발급합니다.
  • 보관: 생성 파일은 내 파일에 최대 3개·24시간 임시 보관 후 삭제됩니다. 업로드 원본은 따로 저장하지 않습니다.

1. 화학 사전보고서

넣는 자료: 실험 매뉴얼 PDF.

  1. 보고서 종류에서 화학 사전보고서를 고릅니다.
  2. 매뉴얼 PDF를 업로드하고, 날짜·학번 등 폼 값을 채웁니다.
  3. 출력 형식(.docx 또는 .hwpx)과 글꼴을 고릅니다.
  4. 생성을 누르면 이론적 배경·원리, 시약 물성, 실험 기구·과정이 학교 양식으로 작성됩니다.

팁: 매뉴얼에 Theory & Principle, 시약 정보가 또렷할수록 결과가 좋습니다. 그림이 필요한 부분은 placeholder와 설명으로 표시됩니다.

2. 화학 결과보고서

넣는 자료: 실험 데이터(엑셀/CSV/텍스트), 실험 사진, (선택) 사전보고서.

  1. 보고서 종류에서 화학 결과보고서를 고릅니다.
  2. 측정 데이터와 사진을 업로드합니다. 데이터 표·그래프 스크린샷도 인식합니다.
  3. 생성하면 평균·표준편차·백분율 오차 등을 계산하고 표·차트를 그려 사전보고서 뒤 결과 분량을 채웁니다.

팁: 직접 정리한 엑셀/CSV가 있으면 우선 반영됩니다. 데이터에 없는 값은 만들지 않습니다.

3. 물리 결과보고서

넣는 자료: PASCO Capstone .cap, 엑셀/CSV/텍스트, 매뉴얼 PDF, 실험 사진 중 하나 이상. 학번은 필수입니다.

  1. 보고서 종류에서 물리 결과보고서를 고릅니다.
  2. .cap 또는 정리한 데이터, 사진을 업로드합니다. 캡스톤의 사용자 입력 표가 있으면 최우선으로 씁니다.
  3. 생성하면 실험 파트별 표·그래프·분석·결론과 오차 분석이 구성됩니다.

팁: .cap 원자료와 직접 정리한 엑셀이 충돌하면 정리 파일을 우선합니다. AI 참고 메모에 실험자 의견을 적어두면 보조로 반영됩니다.

4. 출력 형식과 글꼴

  • .docx: 워드/한글에서 열어 편집. .hwpx: 한글 전용, 표·차트·사진·수식 개체가 직접 들어갑니다.
  • 글꼴은 폼에서 선택합니다. 생성 후 학교 양식·여백을 한 번 더 확인하세요.

5. LaTeX → 한글 수식 변환기

  • 텍스트 변환: 수식 변환기에 LaTeX를 붙여넣으면 한글 수식 스크립트가 나옵니다. 복사해 한글의 입력 → 개체 → 수식(Ctrl+N, M)에 붙여넣으세요.
  • 파일 변환: HWP·HWPX 파일을 올리면 문서 안의 $…$ 형태 LaTeX만 한글 수식 개체로 바꿔 새 .hwpx로 내려받습니다. 모든 처리는 브라우저에서 이뤄집니다.

6. 도구 모음

도구 모음에서 보고서에 자주 필요한 작업을 빠르게 할 수 있습니다(모두 브라우저 전용):

  • 글자수 세기 — 분량 제한 맞추기.
  • 선형회귀·그래프 — 데이터로 추세선·그래프 PNG 생성.
  • LaTeX → 한글 수식 변환기 — 수식을 HWP(HWPX) 수식으로.

자주 묻는 질문

Q. 생성물을 그대로 제출해도 되나요?
아니요. 학습 보조용 초안입니다. 사실·계산·출처를 직접 검토·수정하고, 학교·교사의 AI 사용 기준을 따르세요.

Q. 업로드한 파일은 어떻게 되나요?
보고서 생성에만 쓰이고, 원본은 따로 저장하지 않습니다. 생성 파일만 최대 24시간 임시 보관됩니다. 자세한 내용은 개인정보처리방침을 확인하세요.

Q. 결과에 오류가 있으면요?
내용상 오류가 확인되면 환불정책에 따라 환불 또는 크레딧 복구를 요청할 수 있습니다.

Q. 권한 없는 학교 자료를 올려도 되나요?
안 됩니다. 본인이 이용 권한을 가진 자료만 올리세요. 저작권 책임은 사용자에게 있습니다.
